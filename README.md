# 時間戳記 request & response time
Alpha camp 練習題

## 安裝方式 
1. 開啟終端機，進入至欲儲存此專案的資料夾，並clone此專案
`git clone https://github.com/ChubbyKay/A13_Middleware.git`

2. 切換至專案資料夾
`cd A13_Middleware`

3. 安裝套件
`npm install`

4. 啟動專案
`npm run dev`

5. 若終端機出現`App running on localhost:3000`，請在瀏覽器網址輸入` localhost:3000 `，即可開啟

## 開發環境
・ Node.js v12.18.2<br>
・ Express v4.17.1<br>
